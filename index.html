Quick Start

If you want to get set up quickly, use either of the installation methods below.


Download

master.zip

Install with Bower

bower install hot-chocolate

Getting Started

After installing Hot Chocolate, you need to add hot-chocolate.min.css to your HTML file. To achieve this, simply add the following line of code to the <head> section in your HTML file:


<link rel="stylesheet" href="css/hot-chocolate.min.css">

Alternatively, you can just use the pre-existing index.html file included with Hot Chocolate.

Basic Usage

In Hot Chocolate, there are 12 columns in a grid. If you want to use columns in your layout, you must make sure that the columns are nested under a grid. Here is a basic example of a grid with a three column layout:


            
    <div class="grid">
      <div class="col-4">
        <img src="http://placehold.it/400x400" />
      </div>

      <div class="col-4">
        <img src="http://placehold.it/400x400" />
      </div>

      <div class="col-4">
        <img src="http://placehold.it/400x400" />
      </div>
    </div>
